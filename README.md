# apclattice

## Testing

From the top level of the directory (where `test/` is) run

```python
python -m unittest
```

to run all tests.

## Demo

A demo is included at `examples/demo.ipynb`.

## Documentation

From within the `docs/` directory run

```shell
make html
make latex
```

to build the docs.
